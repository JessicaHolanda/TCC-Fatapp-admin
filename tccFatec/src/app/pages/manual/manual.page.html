Manual do Usuário Fatapp Admin Controlar Salas


  1. No menu, clique na opção “Cadastrar Salas e atividades”;
  2. Para cadastrar uma nova sala clique no ícone de “+”;
  3. Será exibida uma tela onde após o preenchimento e o clique em “Adicionar Sala”;
  4. Será redirecionado a tela anterior, com a nova sala cadastrada;
  5. Para editar a sala, basta clicar no ícone de “lápis” onde será exibido as informações para serem adicionadas à sala.

Cadastrar Recursos


  1. No menu, clique na opção “Cadastrar Salas e Atividades”;
  2. Para editar a sala, basta clicar no ícone de “lápis”;
  3. Clique em “Adicionar Recursos”;
  4. Preencha as informações desejadas e clique em “Adicionar Recurso”;
  5. O sistema redirecionará o usuário para a tela anterior com o recurso cadastrado.

    6. A Figura ao lado representa a interface do Fatapp Admin referente ao percurso para se cadastrar recursos.
Editar Palestrante


  1. No menu, selecione a opção “Editar Palestrante”;
  2. Clique em “Consultar Palestrante”;
  3. Será exibido a tela de pesquisa de palestrante;
  4. No campo pesquisar, pode-se filtrar os palestrantes que já estão cadastrados;
  5. Clique no palestrante que deseja editar;
  6. Será redirecionado a tela anterior com as informações do palestrante para edição ou exclusão, de acordo com a ação desejada, efetue a operação através dos botões “Alterar palestrante” ou “Excluir palestrante”.

    7. A Figura ao lado representa a interface do Fatapp Admin e o passo a passo referente à Tarefa de editar palestrante.
Editar e Excluir Publico Alvo


  1. Na tela de “Cadastrar e editar público-alvo”;
  2. Para editar, clique no “lápis” que está localizado ao lado do público-alvo cadastrado;
  3. Será exibido o formulário com os valores do público-alvo para a edição;
  4. Clique em “Editar Público-alvo” para salvar as informações;
  5. Para excluir, clique no sinal de “-“;
  6. Será exibida a tela de confirmação, e logo após, o registro será excluído do banco de dados.

    7. A Figura ao lado representa a interface do Fatapp Admin e o passo a passo para editar e excluir palestrantes.
Cadastrar nova atividade em uma sala


  1. No menu, selecione a opção “Cadastrar salas e atividades”;
  2. Será exibida as salas cadastradas, clique em “Agendar atividades”;
  3. Será exibido a tela de agenda, que exibe as atividades cadastradas naquela sala e que permite o cadastro de uma nova atividade clicando em “Agendar”;

    4. A Figura ao lado representa a interface do Fatapp Admin e o passo a passo referente à Tarefa de cadastrar uma nova atividade em uma sala.
Editar Atividade
  1. No menu principal selecione “Editar Atividades”;
  2. Clique em “Consultar atividades”;
  3. Será aberto uma tela onde se pode encontrar as atividades cadastradas, e filtrá-las com o campo de Pesquisa;
  4. Após escolher um, clique para continuar a edição de atividades
  5. Clique em “Consultar palestrantes”;
  6. Será aberto uma tela onde se pode encontrar os palestrantes cadastrados, e filtrá-los com o campo de pesquisa;
  7. Após escolher um, clique para continuar a edição de atividades;
  8. Clique em “Consultar eventos”;
  9. Será aberto uma tela onde se pode encontrar os eventos cadastrados, e filtrá-los com o campo de pesquisa;
  10. Após escolher um, clique para continuar a edição de atividades;
  11. Após preencher os demais campos;
  12. Utilize os botões para editar ou excluir a atividade.

    13. A Figura ao lado representa a interface do Fatapp Admin e o passo a passo referente à Tarefa de Editar Atividades.
Gerar QR Code


  1. No menu, selecione a opção “Lista de Atividades”;
  2. Clique no evento que deseja gerar o QR Code;
  3. Será exibido as atividades cadastradas anteriormente;
  4. Clique em “QR Code” em uma das atividades;
  5. Será exibido o QR Code que identifica a atividade proposta, que pode ser impressa e identificada na atividade, para que os alunos possam ler através do aplicativo Fatapp e obter sua presença.

    6. A Figura ao lado representa a interface do Fatapp Admin e o passo a passo para gerar o QR code, que pode ser impresso pelo comando do Windows (Windows + Prt Sc).
Verificar Inscrições nas atividades


  1. No menu, selecione a opção “Lista de presença”;
  2. Clique no evento que deseja visualizar a lista de presença;
  3. Será exibido todas atividades;
  4. Digite uma data válida de início e fim e clique “Filtrar” para visualizar as atividades de um intervalo de datas específicos;
  5. Clique em “Lista de Presença manual” para visualizar a lista dos inscritos na atividade que já marcaram presença.

    6. A Figura ao lado representa a interface do Fatapp Admin e o passo a passo para verificar Inscrições nas atividades.
Encaminhar Lista de presença


  1. No menu, selecione a opção “Lista de presença”;
  2. Clique no evento que deseja enviar a lista de presença;
  3. Será exibido todas atividades;
  4. Clique em Enviar lista de presença por Email;
  5. Será aberto uma tela para digitar o Email ao qual quer enviar a lista;
  6. Clique em Enviar
